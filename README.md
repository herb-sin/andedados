# 📊 Análise de Dados - Monitoramento de Site Público
Trabalho de Análise de Dados UFPI 2º semestre

Este repositório contém um projeto acadêmico desenvolvido com Python e Jupyter Notebook para a disciplina de Análise de Dados, com foco no monitoramento automatizado de dados públicos disponíveis em sites governamentais.

## 🧠 Objetivo

Monitorar a página de datasets do portal [dados.gov.br](https://dados.gov.br/dataset) e extrair informações básicas com o auxílio de técnicas de Web Scraping.

## 📂 Conteúdo do Repositório

- `Untitled.ipynb`: Notebook com todo o código de monitoramento e explicações.
- `andedados.yml`: Arquivo de ambiente Conda com todas as dependências utilizadas no projeto.
- `Trabalho1AnaliseDeDados.pdf`: Versão em PDF do trabalho final (opcional, se aplicável).
